# Identifying Blood Proteomic Markers of Parkinson’s Disease Dementia Using High-Throughput Approaches
This repository contains code for the analyses conducted in "**Identifying Blood Proteomic Markers of Parkinson’s Disease Dementia Using High-Throughput Approaches**"



**Data Availability Statement:**
Proteomics data used in this study is available on the AD Discovery Portal (https://discover.alzheimersdata.org) via application to the Global Neurodegeneration Proteomics Consortium (GNPC, https://www.neuroproteome.org/). 
Phenotypic data from TPD and OPDC cohorts is available through the CPP Integrated Parkinson’s Database (https://c-path.org/tools-platforms/integrated-parkisons-database/). Additionally, data access request can be made directly to the OPDC biosample and data access committee (https://www.dpag.ox.ac.uk/opdc/research/external-collaborations).
